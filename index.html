1st Generation Ryzen CPU Problems

Introduction

Ryzen was introduced to the world on March 2, 2017. It brought an interesting amount of features, that even Intel could not compete with at the time.
At their released, they introduced AMD Ryzen's architechture, the Zen micro architechture. At the same time they released their own concept of Multi-threading, SMT (Simultaneous Multi-threading).
Though as interesting and as innovative these new designs are, comes along the complications of the product such as: Overheating issues, memory incompatibility, SMT support, and the Infinity Fabric latency...

Infinity Fabric and Thermal Problems

The Infinity Fabric is a system of interconnected architechture. The introduction of the Infinity Fabric introduced a CPU that would be efficient, have high core count, and low power, while maintaining a cheap cost.
Though as good as it sounds, the Ryzen CPUs are NOT perfect. The concept is new, and there will be problems that will come with it. The problems that come with the Infinity Fabric are: high latency compared to their competitor's interconnected architecture between each core within the die, and increased dependency on the system's DRAM due to the Zen architecture.
Though there aren't any major problems that come with higher DRAM dependency, ut would be better if the Ryzen CPUs had better compatibility with current generation RAM. Due to this issue, a lot of RAM sticks cannot be accessed with full potential, and RAM cannot be overclocked as high as it's specifications.
As it stands today, Ryzen has trouble completely fixing this issue, but slow progress is being made.
Overheating issues also arise when raising the core clock of the CPU to a higher clock. This is a problem for the CPU because of the lower maximum temperature it has to operate with resulting in higher chances of thermal throttling (A feature that lowers the CPU's speed to cool down), but it isn't a problem with proper cooling, something the stock coolers can't do at higher CPU clock speeds.

The Zen architecture's own SMT is a true multi-threading structure which acts differently depending on its implementation. Multi-threading is used to improve the efficiency of CPUs by using each core to utilize independent threads of execution to better use the resources provided by the processor. Each core and virtual core of the CPU gets an equal share of the workload, however, some cores will require more resources to process the information. Luckily the CPU can do an internal analysis to determine which cores require algorithmic priority. However, when running SMT with only 1 thread, the other threads can be a bottleneck and slow the system down as the cores are idle half of the time with statically partitioned cores giving them equal timings with the thread that is handling the workload. More problems arise regarding Windows 10, Ryzen is a new concept and Windows could not apply and create stable compatibility with Ryzen. Windows slight bottlenecks that treat virtual cores from Ryzen SMT into primary cores, this can result in workloads meant for REAL primary cores to be given to virtual cores, resulting in a bottleneck.

Summary of 1st generation AMD Ryzen problems

1. Overheating Issues/Low Max Temp

  • Low maximum temperature, resulting in higher chances and lower temperature to trigger thermal throttling.
  • Overclocking is harder to do and maintain.
  • System will shutdown early because of lower max temp.

2. Memory Incompatibility

  • System RAM cannot be overclocked to its full potential, and in some cases, can cause crashes with poor compatibility RAM.
  • Poor RAM compatibility can cause a the CPU to slow down as the CPU relies on the RAM especially on Ryzen CPUs

3. SMT Support

  • Each core is equally statically partitioned, with equal timings causing bottlenecks. Virtual cores are treated as Primary cores, increasing workload for virtual cores.
  • Most of today's applications especially games cannot fully utilize Simultaneous Multi-Threading as they are built for CPUs 4 cores or less.

4. Infinity Fabric

  • Higher latency compared to competitors interconnected architecture, higher dependency on RAM. RAM incompatibility can cause more problems due to this.
